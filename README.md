# lisker-blockchain-demo
This demo shows how to implement Lisker in your client application and communicate with a Lisk sidechain containing the Lisker module.

It's build with react and Lisk elements. And makes use of the following:

-  [Lisker Avatar Generator](https://github.com/Korben3/lisker-avatar-generator)
-  [Lisker Avatar Viewer](https://github.com/Korben3/lisker-avatar-viewer)
-  [Lisker Blockchain Module](https://github.com/Korben3/lisker-avatar-module)

### Prerequisites

- Node.js

### Installation

```
git clone https://github.com/Korben3/lisker-blockchain-demo
cd lisker-blockchain-demo
npm install
```

### Config

You can change the node the client is connected to in `src\config\config.json`

### Run

In the client directory run: `npm start`
