# lisker-blockchain-demo
This demo shows how to implement Lisker in your client application, build with react and Lisk elements.

### Prerequisites

- Node.js

### Installation

```
git clone https://github.com/Korben3/lisker-blockchain-demo
cd lisker-blockchain-demo
npm install
```

### Config

You can change the node the client is connected to in `src\config\config.json`

### Run

In the client directory run: `npm start`
